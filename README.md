# Superstore Sales Executive Dashboard 📊

An interactive Executive Sales Dashboard built using **Microsoft Power BI** to analyze business performance, track key metrics, and forecast future sales trends.

---

## 🚀 Project Overview
This project provides a comprehensive analysis of the Superstore dataset. It is designed to give executives and stakeholders clear visibility into sales performance, profitability, product category performance, and geographic distribution across the United States.

---

## 📈 Key Performance Indicators (KPIs)
* **Total Sales:** $14.92M
* **Total Quantity:** 215K
* **Total Profit:** $1.52M
* **Total Orders:** 5K

---

## 🛠️ Features & Insights
* **Sales Forecast:** Integrated a predictive analytics line chart to forecast future sales trends.
* **Geographic Analysis:** Interactive US Map visual tracking sales distribution by state.
* **Category Breakdown:** Detailed Profit analysis by product categories (Technology, Office Supplies, Furniture).
* **Top Products:** Highlighting top-performing products by total sales.

---

## 💻 Tools & Technologies
* **Microsoft Power BI** (Data Modeling, DAX, Power Query)
* **Visualizations:** Maps, Line Charts with Forecasting, Bar Charts, and KPI Cards.
